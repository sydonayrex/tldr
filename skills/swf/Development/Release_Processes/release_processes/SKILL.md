# Release Process

## Release Checklist
- [ ] All tests pass
- [ ] Security scan clean
- [ ] Performance within budget
- [ ] Release notes written
- [ ] Rollback plan documented
- [ ] Monitoring dashboards ready
- [ ] On-call notified
- [ ] Stakeholders informed

## Self-Correction
- Release failed? Execute rollback, post-mortem
- Missing checklist item? Add to template
- Coordination gap? Improve communication process
