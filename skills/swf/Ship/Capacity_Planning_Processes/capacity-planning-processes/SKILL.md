# Capacity Planning Process

## Monthly Capacity Review
1. Current utilization vs thresholds
2. Growth rate trend
3. Known upcoming events (营销 campaigns, product launches)
4. Scaling actions needed this month
5. Cost projection

## Quarterly Deep Dive
1. 6-month forecast
2. Reserved instance / committed use planning
3. Cost optimization opportunities
4. Architecture changes for scale

## Self-Correction
- Capacity exceeded? Emergency scale, improve forecasting
- Over-provisioned? Right-size, reduce waste
- Forecast consistently wrong? Improve model/data
